(1) start <= end
(2) (start + end) // 2
(3) a[middle] == key
(4) middle - 1
(5) middle + 1
이진 검색 알고리즘 (Binary Search Algorithm)

이진 검색이란 데이터가 정렬되어 있는 배열에서 특정한 값을 찾아내는 알고리즘이다.
배열의 중간에 있는 임의의 값을 선택하여 찾고자 하는 값 key와 비교한다.
key가 middle 보다 작으면 middle을 기준으로 좌측의 데이터들을 대상으로,
key가 middle보다 크면 배열의 우측을 대상으로 다시 탐색한다.
동일한 방법으로 다시 중간의 값을 임의로 선택하고 비교한다.
해당 값을 찾을 때까지 이 과정을 반복한다.

이진 검색 알고리즘의 시간 복잡도는 O(logN)이다.
